# Contributing

Contributions are welcome if they help preserve interesting questions and improve the quality of answers.

## How to Add a Question

1. Choose the yearly file under `questions/`.
2. Copy the format from `TEMPLATE.md`.
3. Write one clear question.
4. Add the best answer you can currently provide.
5. Mark the status honestly:
   - `Answered`
   - `Best current answer`
   - `Open mystery`

## Answer Quality

Good answers should be clear, cautious, and easy to revisit later. If a question has no settled answer, do not pretend it does. Write what is known, what is uncertain, and what could be checked next.

## Style

- Questions can come from any field.
- Short entries are fine.
- Strange questions are welcome.
- Avoid unsupported certainty.
- Add sources when they are useful.
